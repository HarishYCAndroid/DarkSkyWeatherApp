 Steps for Running the application

1) Unzip the file.
2) Import the project into Android studio.
3) Make sure your version of SDK and Gradle is supporting latest (Grade version : 2.3.2 & Compiler SDK version : 25).
4) Once project is imported with errror Free. Click on Run button.
5) Emulator or device will open screen with output.
6) Attached Image for reference.

Concepts Used in Application :

1) Used Third party libs (Async Http Client) - To make async http calls for request
2) Used GSOn - for Json parsing and coverting to Json object
3)Recyler View - for binding dynamic data.
4)Shared Preference - For data persistnece in client.
